# 更新记录

## v1.0.0 — 2026-07-19

### 新增

- 新增风扇温控、热点管理和设备管理 LuCI/网页界面及测试。
- 新增 OpenClash UI 修补、离线恢复、只读核验和稳定基线恢复工具。
- 新增 Netdata v1.38.1 简体中文覆盖层，支持 LuCI 内嵌页面、批量部署、自动备份和回滚。
- 新增 RST 恢复、完整备份、DNS 分流、硬件与供电、设备监控及 Netdata 部署文档。

### 修改

- 将 Netdata 验收目标统一为 LuCI 内嵌页面，加入 iframe 与资源版本参数，避免浏览器继续使用旧缓存。
- 统一风扇默认温度曲线为 `30/35/35/38/37/40°C`，并保留可重复部署和验收脚本。
- 补充部署前备份、失败回滚、批量生产和 fresh 黑盒测试要求。
- 扩充 README，明确模块入口、验证命令和安全边界。

### 修复

- 修复 Netdata 中文覆盖层未在 LuCI iframe 中生效的问题。
- 修复动态、分段文本仍显示英文的问题，并隐藏 Cloud、登录、定价和社交推广入口。
- 修复 Netdata 手动回滚路径可接受异常字符的问题。
- 修复多项恢复脚本和验证工具的默认地址、路径及失败处理问题。
- 修复恢复固件缺少 SFTP 子系统时热点部署无法上传的问题，改用 SSH 标准输入传输文件。
- 修复热点扫描调用 JSON 解析函数前未加载 `jshn.sh` 的问题。

### 移除或不再发布

- 从公开仓库和 Release 备份中排除设备完整镜像、MTD/eMMC 转储、配置归档和历史操作备份。
- 排除 `docs/私密/`、本地密码辅助脚本、代理订阅、Wi-Fi/管理凭据和密钥。
- 排除 `.tmp/`、`tmp/`、缓存、`.bak`、安装包及其他可再生成文件。
- 不发布不兼容 Netdata v1.38.1 的上游旧 dashboard 文件和安装器，只保留来源说明与已验证的 `compat/` 实现。

### 安全

- 扩展 `.gitignore`，覆盖敏感目录、大体积设备产物、凭据文件和发布临时文件。
- 公开发布只保留源码、公开文档、测试与预览资源；备份资产在发布前再次执行敏感信息扫描。
