# Banana Pi R3 Mini

本项目用 [dd](https://github.com/JustLikeCheese/dd) 管理规约。

## 冷启动（agent 必读，不可跳）

1. 阅读完整的 `docs/必读/`。
2. 加载 `/dd`，并按需运行 `dd get <TAG>`。
3. 商家交付的设备参数与敏感凭据存放在本机私密文档 `docs/私密/商家交付信息.md`；仅在确有配置需求时读取，不得提交、打印或转发。

## 项目必读清单

- DOC（文档编写）
- TEST（测试流程）
- CODE（项目编写）
- MEM（记忆规约）

## dd 集成

项目没有额外 tag skill。运行 `dd list` 查看全局触发字，或运行 `dd get <TAG>` 读取所需规约。
